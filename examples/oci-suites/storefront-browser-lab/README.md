Storefront Browser Lab

Browser-first commerce suite with Kafka event streams, Playwright checkout journeys, and mock APIs for catalog and order flows.

Structure

- `suite.star`: declarative topology
- `profiles/`: launch profiles
- `api/`: OpenAPI and protobuf contracts
- `mock/`: deterministic mock responses
- `scripts/`: bootstrap hooks
- `scenarios/`: Playwright scenarios
- `fixtures/`: seeded products and users
- `policies/`: lightweight validation policies
